# 师说新语
- 本项目是一个拥有能为教师智能备课、生成教案、设置提醒、管理学生等功能的web端平台，由电子科技大学“极客先锋”小队于2025-2完成
- 项目现已软著送审：流水号：2025R11L0810290
- 机器学习源码：https://github.com/Yaro1214/AI_Teacher
- 后端源码：https://github.com/Vainseem/TeachGenie-BE

### 前端技术栈
- vue3、VueCLI
- pinia
- element plus
- ECharts
- Sass
- marked（一个展现MarkDown文本格式的库）
- websocket(与AI对话建立长连接，便于文本的流式输出)

### 内置功能
1. web端的登陆、注册、忘记密码等基本功能
2. AI智能备课助手功能：与AI对话，通过websocket长连接与机器学习端一键生成教师所需要的备课方案、课后习题并使用marked库进行MarkDown格式的渲染
3. 资源库功能：展示所有用户共同上传的教学备课资源，如PPT、视频等内容皆可以上传至此
4. 工作台功能：展示教师的数据统计、课程表、代办提醒、教学任务以及学生反馈，让用户添加代办提醒、教学任务
5. 课程表功能：让教师在此自定义课程安排
6. 学生管理功能：ECharts表格动态可视化展现学生的学习成绩波动、学习成绩构成、知识点掌握情况等
7. 账号设置功能：修改个人信息
